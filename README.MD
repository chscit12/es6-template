# Template for using compiled ES6 on client and server

## Install

```git clone https://github.com/chscit12/es6-template```

```npm install```

## Usage

### Start the server

```npm run start```

### Start webpack

```npm run webpack```

## Dependencies

testet on: 

* Node v7.0.0
* NPM 3.10.8

## Description

Use this template in case you want to use es6 syntax on server and client side
